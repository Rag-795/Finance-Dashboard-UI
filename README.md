# Finance Dashboard

A stunning finance tracking dashboard built with React and Tailwind CSS v4.

## Features

- **Pristine UI/UX:** Clean layout with a floating capsule navbar, glassmorphism overlays, and fully responsive design.
- **Smart Data Visualization:** Beautiful, accessible charts rendered with Recharts.
- **Role-Based Access:** Toggle between Admin and Viewer modes to simulate permission boundaries.
- **Micro-interactions:** Silky smooth animations powered by GSAP and native CSS transitions.
- **Dark Mode Support:** Seamless toggling between light and dark themes.

## Tech Stack

- **Framework:** React 18 (Vite)
- **Styling:** Tailwind CSS v4, Lucide React (Icons)
- **State Management:** Zustand
- **Routing:** React Router
- **Animations:** GSAP
- **Data Visualization:** Recharts

## Local Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
